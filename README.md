<h1 align="center">Vishnu</h1>
<p align="center">A simple and minimalist toolkit with modern css</p>

## Install

**Using npm**

```sh
$ npm install --save vishnucss
```

**Using directly link**

```html
<link rel="stylesheet" href="https://unpkg.com/vishnucss"/>
```
